===============================
 How to Use 1-my_list.py
===============================

The MyList class inherists from list and have a function to print a sorted list

Numbers
========
::

    >>> MyList = __import__('1-my_list').MyList

    >>> my_list = MyList()
    >>> my_list.append(2)
    >>> my_list.append(12)
    >>> my_list.append(2.4)
    >>> my_list.append(-2)
    >>> my_list.append(0)
    >>> print(my_list)
    [2, 12, 2.4, -2, 0]
    >>> my_list.print_sorted()
    [-2, 0, 2, 2.4, 12]
    >>> print(my_list)
    [2, 12, 2.4, -2, 0]

    >>> list1 = MyList()
    >>> list1.append(None)
    >>> print(list1)
    [None]
    >>> list1.print_sorted()
    [None]

Errors
======

    >>> my_list = MyList()
    >>> my_list.append()

    TypeError: append() takes exactly one argument (0 given)
